Welcome to my blog

How I Started Coding

My journey into the world of coding began at a young age. I was always fascinated by computers and how they worked. I started by tinkering with basic HTML and CSS, creating simple web pages for fun.

As I grew older, I realized that coding was not just a hobby, but a valuable skill that could open up many opportunities. I decided to take it more seriously and enrolled in online courses to learn programming languages like Python and JavaScript.

Through dedication and practice, I gradually improved my skills and started working on real projects. I found joy in solving problems and building applications that could make a difference.

Today, I am proud to say that I am a confident coder, and I continue to learn and grow every day. Coding has not only become a career for me but also a passion that I cherish.

Understanding HTML

From Curiosity to Code: My Journey Into HTML

If you told me a year ago that I’d be writing code, I probably would’ve laughed. I wasn’t the “techie” type, and I didn’t even know what HTML stood for. But one day, something clicked — and that click came from curiosity.

It All Started With a Simple Question

I remember browsing a website and wondering: How does this even work? Everything looked clean — images, text, buttons — but I had no idea how it all came together behind the scenes. That innocent question became a rabbit hole. I Googled how websites are built, and the first thing I kept seeing was something called HTML,” they said — the skeleton of every webpage.

Learning the Basics

I opened up freeCodeCamp and started the HTML module. At first, it was strange: tags like felt like learning a new language.

But as I practiced, things started to make sense.

  • <h1> That's a heading.
  • <p> A paragraph.
  • <img>Image tag.
  • <a href> A link.

I realized HTML isn’t hard — it’s actually like telling the browser what each part of your content is. And once you get that, a whole new world opens up.

What i know now

Today i can,

  • Structure entire pages with semantic HTML
  • Use forms, lists, tables, and multimedia tags
  • Keep my HTML clean and accessible

More importantly, I understand that HTML isn’t just code — it’s communication. It tells the browser what your content is, and tells assistive tech (like screen readers) how to present it to others.

What's Next?

After HTML came CSS, then JavaScript. But I’ll always remember that first spark — learning how to write a simple line of code that changed the way I see the web.
So if you’re just starting out and feel overwhelmed: don’t. Open your code editor. Type a line. Save the file. Open it in a browser.
That’s how it starts.
